# 透明背景

不知道为什么会被叫做幽灵`Ghost`。使用`dashed`变成虚线

```html
<cg-space>
  <cg-button ghost>Default</cg-button>
  <cg-button type="primary" ghost>Primary</cg-button>
  <cg-button type="info" ghost>Info</cg-button>
  <cg-button type="success" ghost dashed>Success</cg-button>
  <cg-button type="warning" ghost dashed>Warning</cg-button>
  <cg-button type="error" ghost dashed>Error</cg-button>
</cg-space>
```
