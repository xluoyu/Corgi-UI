# 按钮 Button

可以触发某些操作

```demo
type
color
circle
disabled
size
dashed
```

# props

| 名称 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| attr-type | `button` | `submit` \| `reset` \| `button` | 按钮的 DOM 的 `type` 属性 |
| block | `boolean` | `false` | 按钮是否显示为块级 |
| bordered | `boolean` | `true` | 按钮是否显示 border |
